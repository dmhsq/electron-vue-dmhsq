# electron-vue-dmhsq

## 安装依赖
```
npm install
```

### 热开发页面
```
npm run serve
```

### 只打包vue 不打包electron
```
npm run builds
```
### 热开发页面
```
npm run build
执行完成后
开启额外的控制台
cd electron
npm run serve
```

### 热开发electron
```
npm run builds
执行完成后
开启额外的控制台
cd electron
npm run serve
```

### 打包electron应用
```
在electron文件夹中
npm run build
```
